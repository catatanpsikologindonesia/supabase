# Repository Map

## Top-Level Areas

- `supabase/`
  Supabase CLI workspace, migrations, functions, and local config.
- `scripts/`
  Automation for start, restore, pull, mirror, verify, sync, deploy, and guards.
- `snapshot/`
  Database, function, storage, and verification artifacts.
- `integrations/gas-mail-dispatcher/`
  Google Apps Script mail bridge.
- `knowledge/`
  Repo documentation.
- `.agents/skills/project-context/`
