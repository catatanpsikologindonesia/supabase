# Language Policy

Last updated: 2026-05-20

## Documentation Language

All `.md` files in this repository must be written in English without exception.
This includes: CLAUDE.md, AGENTS.md, knowledge docs, changelogs, runbooks, and migration notes.

## UI Language - Backend

No UI strings exist in this repository. Language policy applies to documentation only.

## What Is Excluded

- TypeScript/JavaScript variable names, type names, function names
- Route path strings
- Code comments
- Console log messages
- HTML `class`, `id`, `data-*` attributes

## Enforcement

Before committing any `.md` changes, verify no Indonesian text has been introduced.
